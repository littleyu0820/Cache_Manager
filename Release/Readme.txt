==============================
Cache Manager v1.0
==============================

📌 介紹：
Cache Manager 是一個小型工具，可以記錄您使用應用程式的時間，
並提供統計與快捷啟動功能，幫助您更高效地管理日常使用習慣。

🖥️ 使用方式：
1. 解壓縮本壓縮檔到任意資料夾。
2. 雙擊執行 "CacheManager.exe" 即可開始使用。

📁 資料儲存：
- 程式會自動儲存使用記錄與自定義資訊於同資料夾下的 JSON 檔案中。
- 若要重置所有紀錄，只需刪除這些 JSON 檔案即可。

📦 注意事項：
- 本程式為 Windows 桌面應用，無需額外安裝其它程式。
- 請勿移除資料夾中的 `platforms/` 或 DLL 檔案，否則程式將無法啟動。
